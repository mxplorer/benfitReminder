# Brand Assets

- `tray-source-1024.png` — **MASTER** line art. 1024×1024, transparent bg,
  black card+bell strokes. Commit this as-is; everything else is derived.
- `ccb-logo-1024.png` — **DERIVED**. Silver-metallic squircle + dark-navy line
  art, generated by `npm run build:desktop-icon`. Consumed by
  `npm run tauri icon assets/brand/ccb-logo-1024.png` to produce the full
  icon set under `src-tauri/icons/`.

## Regeneration flow

After editing `tray-source-1024.png`:

```
npm run build:desktop-icon     # → assets/brand/ccb-logo-1024.png
npm run tauri icon assets/brand/ccb-logo-1024.png
npm run build:tray-icons       # → src-tauri/icons/tray/*.png
```

## Re-squaring a non-square source

If `tray-source-1024.png` arrives non-square from a design tool, run
`node scripts/prep-brand-assets.mjs` to pad it to 1024×1024 in-place
(transparent padding).
